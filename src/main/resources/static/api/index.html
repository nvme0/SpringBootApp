Topics Route

Route: /api/topics

  • GET /api/topics Gets all topics
  • GET /api/topics/{id} Gets a topic by id
  • POST /api/topics Create a new topic
  • PUT /api/topics/{id} Updates a topic
  • DELETE /api/topics/{id} Deletes a topic
